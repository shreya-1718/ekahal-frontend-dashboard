# Product Dashboard

## Tech Stack

- Next.js 15
- TypeScript
- Tailwind CSS
- React Context API
- DummyJSON API

## Features

- Product listing
- Search
- Category filtering
- Favorites with localStorage
- Product detail page
- Responsive design

## Getting Started

npm install

npm run dev

## Architecture

- Server Components for data fetching
- Client Components for UI interactions
- Context API for global favorites
- Service layer for API calls
